/* $DOC$
   $FUNCNAME$
      FIELDTYPE()
   $CATEGORY$
      Database Tools
   $ONELINER$
      Determines the type of a given field.
   $SYNTAX$
      FIELDTYPE(<nFieldNum>) --> cFieldType
   $ARGUMENTS$
      <nFieldNum> Data field , which type need to be determined.
   $RETURNS$
      FIELDTYPE() returns the character that designates the type of
      a given field:</par>
      <table>
      'C'   character string;
      'N'   numeric;
      'L'   logical;
      'D'   date;
      'M'   memo.
      </table>
   $DESCRIPTION$
      This function determines the type of a field, designated by its
      number.
   $EXAMPLES$
      PROCEDURE Main()
         LOCAL i
         USE tests NEW
         FOR i := 1 TO FCount()
            ? FieldType( i )
         NEXT
         USE
         RETURN
   $STATUS$
      R
   $COMPLIANCE$
      This function is CA-Cl*pper tools compatible
   $FILES$
      Library is libmisc
   $SEEALSO$
      FIELDSIZE(),FIELDDECI()
   $END$
 */
